# Проект: Место

### О проекте
 Сервис Mesto представляет собой интерактивную страницу, куда можно добавлять фотографии, удалять их и ставить лайки.

### Навыки

* HTML
* CSS
* GRID
* FLEX
* JavaScript
* Адаптивная верстка
